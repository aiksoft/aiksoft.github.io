(FancyText 1.0 GUI)

FancyText(FT) is a free, simple and easy-to-use desktop application designed basically to be used to create pictures containing fancy (beautiful and elegant) text graphics as well as other graphics such as pictures, simple shapes such as line, circle, square, polygon, etc, complex shapes, which are created from simple shapes, and emojis. The pictures can be saved in JPG, JPEG, GIF or PNG format.

FancyText is written entirely in the Java programming language.

An outstanding feature of FancyText is the number of available fonts that the user has at its disposal. FancyText comes packaged with eight hundred and sixty one (861) custom fonts in addition to the fonts which are locally installed on the user's machine, which is usually above two hundred on most computers. Consequently, the user can have well over a thousand fonts to select from when creating beautiful text graphics using FancyText.

Font preview

Another cool feature of FancyText is the ability to add effects to the text and other graphics. The software has about thirteen (13) basic effects which include, MotionBlur, Lighting, ColorAdjust, Reflection, Perspective, etc., which are fully adjustable and can be applied alone or can be combined to give a variety of very beautiful and complex effect. This feature will also be improved upon in subsequent updates and versions of the software.

Another yet cool feature of FancyText is the ability to add various kinds of emoticons and emojis to your post. This feature will also be periodically updated to make your post more elelegant and give a more vivid expression to your idea or the message you are trying to communicate.

(Emojis Dialog Screenshot)

FancyText was designed to be simple and very easy to use. With FancyText your fancy photo could be ready in a matter of minutes depending on your familiarity with the software and your creativity. Overall when it comes to using FancyText, your creativity is your only limit.

Standalone Application Download

The Standalone version of the application can be run in any computer (Windows, Mac, Linux, Unix, etc.) which has the appropriate Java Runtime Environment (JRE) installed. See What is the JRE? to learn more about what is a JRE. Click here to find out if your system already has the JRE installed. Please note, if your system already has the JRE installed, BUT the version is less than 1.8.0_141 (i.e. version 8, update 141), you need to update or reinstall the latest version of the JRE. Click here to download latest version of the JRE. Click here for technical assistance on how to install and configure the JRE.

The main benefits of the Standalone version are its portability and smaller download size. However, if you don't have the JRE installed on your machine you need to download it from the official website of Oracle, the vendor of the Java programming language. Click here to download latest version of the JRE. (Note: Please ensure that you download the ".exe" version of the JRE installer if you are not an advanced computer user.)

Click here to download the zip file of the stanalone version of the application.

After downloading the zip file containing the standalone version of the application, unzip it and make sure the lib folder which contains the preloader for the application and the jar file are in the same folder after unzipping the file . Double click the executable FancyText.jar file to load and use the application.
